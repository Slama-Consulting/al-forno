📋 Comment mettre en ligne sur GitHub Pages (gratuit)

1

Créer un compte GitHub

Va sur github.com et crée un compte gratuit si tu n'en as pas.

2

Créer un nouveau dépôt

Clique sur New repository. Donne-lui un nom simple, ex : alforno-avis. Coche "Add a README file". Clique Create repository.

3

Uploader le fichier avis.html

Dans ton dépôt, clique Add file → Upload files. Glisse le fichier avis.html que tu as téléchargé. Clique Commit changes.

4

Activer GitHub Pages

Va dans Settings → Pages. Sous "Source", choisis Deploy from a branch → main → / (root). Clique Save. Attends ~1 minute.

5

Ton URL sera :

https://TON-USERNAME.github.io/alforno-avis/avis.html

Remplace TON-USERNAME par ton nom d'utilisateur GitHub.

💡 Conseil : Le fichier al-forno-avis.html (avec le dashboard) reste sur ton ordinateur — tu ne le mets PAS sur GitHub. Seul avis.html est public.
